# Yoneis Optimizer

**A powerful performance optimizer mod for Rust.**

Yoneis Optimizer is a client-side tool that helps you boost your FPS in Rust Legacy by giving you real-time control over the most performance-heavy settings, especially **Terrain** (trees, grass, and LODs).

![Rust Yoneis]((https://github.com/TrecTar/yoneis/raw/main/banner.png))

## Features

- **Clean and intuitive GUI menu** (press **F2**)
- **Advanced FPS Counter** with Min / Current / Average (press **F3**)
- Real-time adjustment of:
  - Tree Distance
  - Tree Billboard Distance
  - Tree Crossfade Length
  - Tree Maximum Full LOD Count
  - Heightmap Pixel Error
  - Heightmap Maximum LOD
  - Basemap Distance
  - Detail Distance (Grass)
  - Detail Density
  - Master Texture Limit
  - Render Distance
- **One-click Recommended Settings** button
- Quick Grass ON/OFF buttons
- Persistent loader using `DontDestroyOnLoad`

## Hotkeys

- `F2` → Toggle Optimizer Menu
- `F3` → Toggle FPS Counter

## Installation

1. Download the latest release here: [**Download Yoneis v0.5**]
2. Replace the DLL in your game's `RustYoneis_Data/Managed` folder
3. Launch the game
4. Join a server and press **F2**
